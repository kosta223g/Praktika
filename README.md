# Practice 1 — FastAPI + uv

Базовий FastAPI-додаток, створений у межах практичної роботи №1.
Проєкт використовує **uv** для керування залежностями та віртуальним середовищем,
а також **Ruff** і **pre-commit** для автоматичної перевірки якості коду.

## Вимоги

- Python 3.12+
- [uv](https://docs.astral.sh/uv/getting-started/installation/)

## Встановлення

1. Клонуйте репозиторій:
   ```bash
   git clone <URL-вашого-репозиторію>
   cd practice1-fastapi
   ```

2. Встановіть залежності (uv автоматично створить віртуальне середовище):
   ```bash
   uv sync
   ```

## Запуск проєкту

Запустіть сервер розробки FastAPI:

```bash
uv run fastapi dev main.py
```

Після запуску застосунок буде доступний за адресами:

- API: http://127.0.0.1:8000/
- Інтерактивна документація (Swagger): http://127.0.0.1:8000/docs

## Перевірка якості коду

Проєкт використовує Ruff (лінтер + форматер) через pre-commit хуки.

Встановити хуки (виконується один раз після клонування):

```bash
uv run pre-commit install
```

Запустити перевірку вручну на всіх файлах:

```bash
uv run pre-commit run --all-files
```

## Структура проєкту

```
.
├── main.py                   # Точка входу FastAPI-застосунку
├── pyproject.toml            # Конфігурація проєкту та залежності
├── uv.lock                   # Зафіксовані версії залежностей
├── .pre-commit-config.yaml   # Конфігурація pre-commit хуків
└── .gitignore
```
